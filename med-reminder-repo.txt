## ğŸ“¦ Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/medicine-reminder.git
cd medicine-reminder
```

### 2. Hardware Setup

1. Attach the Waveshare 2.13" e-Paper HAT to your Raspberry Pi Zero 2 W
2. Insert microSD card with Raspberry Pi OS
3. Connect power supply

### 3. Software Installation

```bash
# Run the automated setup script
chmod +x setup.sh
./setup.sh
```

Or follow the [Manual Installation Guide](docs/INSTALLATION.md)

### 4. Configuration

Edit `config.json` to customize your medication schedule:

```json
{
  "schedule": [
    {
      "name": "Morning Medicine",
      "hour": 7,
      "minute": 45
    },
    {
      "name": "Night Medicine", 
      "hour": 22,
      "minute": 0
    }
  ],
  "timezone": "America/New_York",
  "alertWindow": 300
}
```

### 5. Deploy

**Option A: Web-Based (Recommended)**
```bash
# Upload index.html to your hosting provider
# Update KIOSK_URL in setup.sh
./deploy-web.sh
```

**Option B: Local Python**
```bash
python3 src/medicine_reminder.py
```

## ğŸ› Troubleshooting

### Display Not Working
```bash

```

### Time Zone Issues
